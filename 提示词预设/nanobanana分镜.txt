- Maintain perfect visual continuity across all panels: same people, same clothes, same lighting, same

Style:- Cinematic, photorealistic, film-like quality 
- Consistent color grading and lighting matching the reference image 
- High detail, sharp where focused, natural film grain if present in reference 
Final output: 
- Generate and show ONLY the single contact sheet grid image (no text explanations unless asked). 
- If you add any text, keep it minimal and only as labels on the panels.
